# coreKit

任何 go-zero 框架项目，都可以直接将 coreKit 作为依赖引入，从而使用 coreKit 提供的功能。
因此，在这个目录下，添加功能时，请确保这些功能对 go-zero 框架是通用的，且最好不要添加与具体业务强相关的功能，除非你能够明确想要魔改。

## 使用方式

1. 直接复制整个 `coreKit` 目录到你的项目根目录中
2. 将包引用路径替换为你项目中 `go.mod` 文件中的 module 名称
3. 去 `coreKit/constant.go` 文件中，适当修改常量的值，以符合你项目的需求
4. 在你的项目根目录下运行 `go mod tidy`，以确保所有依赖都正确引入
